Apresentação

Seja bem-vindo!!

O SpeedyBurger surgiu com a missão de proporcionar uma experiência singular e servir alimentos de qualidade. O objetivo é levar nossos produtos ao máximo de clientes possível sempre prezando por nossos três pilares fundamentais:

  1. Agilidade na entrega
  2. Ingredientes de alta qualidade
  3. Preço justo

Objetivos

Este site tem o objetivo de apresentar a você a empresa SpeedyBurger. Aqui você poderá ver as últimas notícias da empresa, consultar nosso cardápio, fazer uma solicitação de pedido, conhecer nossas redes sociais, entrar em contato conosco e ver onde estão localizadas as nossas lojas!

Público alvo

Pessoas de todas as idades que procuram uma boa lanchonete Fast Food!!

Tecnologias utilizadas

Foram utilizadas as seguintes tecnologias para a construção do site:

  • Vs code e suas extensões
  • Navegadores mais populares
  • Ferramentas do desenvolvedor em navegadores
  • simulação de dispositivos móveis
  • Font Awesome para ícones
  • GitHub

Restrições

Este site não funcionará corretamente no navegador internet explorer ou em telas com menos de 295px de largura.

Requisitos mínimos

Para acessar o site você deve ter:

  • Uma conexão com a internet
  • Um dispositivo eletrônico capaz de acessar a internet
  • Navegador de internet atualizado, de preferência Google Chrome

Navegadores compatíveis:

  • Google Chrome - Versão 69 ou superior
  • Mozilla Firefox - Versão 83 ou superior
  • Microsoft Edge - Versão 79 ou superior
  • Opera - Versão 64 ou superior

Dificuldades encontradas

As maiores dificuldades para a construção do site foram:

  • Pensar e montar o design
  • Fazer a compatibilidade com outros navegadores
  • Adaptar o site para dispositivos móveis

Pontos fracos e fortes

Periodicidade de atualização

Todos os dias uma parte do código fonte era melhorada para que se chegasse no melhor resultado possível, depois de finalizado a atualização seria para as notícias ou para quando alguma coisa mudar na empresa, como inclusão de novos lanches.

Aplicação do bootstrap

O Bootstrap não foi utilizado na construção desse site.
